
pirates is an attempt in creating a ray-traced based renderer suitable for video games or other interactive real-time applications. As such its main goal is maintaining at least 30 fps on average. Planned rendering features are:

-reflection

-refraction

-soft shadows 

-global illumination

-physically based rendering
